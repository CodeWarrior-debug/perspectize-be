  <action>
    Create SvelteKit project in `fe/` directory (sibling to backend/):

    1. Run project creation:
       ```bash
       npx sv create fe --template minimal --types typescript --no-add-ons
       cd fe
       ```

    2. Install adapter-static and configure svelte.config.js:
       ```bash
       npm i -D @sveltejs/adapter-static
       ```
       Configure adapter with: pages: 'build', assets: 'build', strict: true

    3. Add Tailwind CSS v4:
       ```bash
       npx sv add tailwindcss
       ```

    4. Initialize shadcn-svelte (interactive - select: TypeScript yes, Tailwind v4 yes, default style):
       ```bash
       npx shadcn-svelte@latest init
       ```

    5. Add Button component to validate shadcn setup:
       ```bash
       npx shadcn-svelte@latest add button
       ```

    6. Configure custom navy theme in src/app.css:
       - Override --primary with navy: oklch(0.216 0.006 56.043) which is #1a365d
       - Override --primary-foreground: oklch(0.985 0.001 106.423) for contrast
       - Keep other shadcn defaults

    7. Add prerender = true to +layout.svelte for SSG:
       ```svelte
       <script lang="ts">
         export const prerender = true;
       </script>
       ```

    IMPORTANT: The project must be created at repository root level as `fe/`, NOT inside backend/.
  </action>
  <verify>

    ```bash
    cd fe && npm run dev
    ```
    Dev server starts on http://localhost:5173 without errors.

    Check that src/lib/components/ui/button/ directory exists (shadcn Button installed).
  </verify>

  <files>

  <action>
    1. Download Inter variable font and place in static/fonts/:
       ```bash
       mkdir -p fe/static/fonts
       curl -L "https://github.com/rsms/inter/raw/master/docs/font-files/InterVariable.woff2" -o fe/static/fonts/Inter-Variable.woff2
       ```

    2. Add font-face and preload to app.css and app.html:

       In app.css add:
       ```css
       @font-face {
         font-family: 'Inter';
         src: url('/fonts/Inter-Variable.woff2') format('woff2');
         font-weight: 100 900;
         font-display: swap;
       }
       ```

       In app.html head add:
       ```html
       <link rel="preload" href="/fonts/Inter-Variable.woff2" as="font" type="font/woff2" crossorigin>
       ```

       Update @theme in app.css:
       ```css
       @theme {
         --default-font-family: 'Inter', system-ui, -apple-system, sans-serif;
       }
       ```

    3. Create folder structure with .gitkeep files:
       - src/lib/components/ (flat, all components here - but ui/ subfolder from shadcn is OK)
       - src/lib/queries/ (GraphQL queries and mutations)
       - src/lib/utils/ (utility functions - cn.ts already exists from shadcn)
