# Perspectize Frontend

Svelte-based web application for the Perspectize platform.

## Tech Stack (Planned)
- **Framework:** Svelte/SvelteKit
- **State Management:** TanStack Query
- **Data Grid:** AG Grid
- **GraphQL Client:** URQL or Apollo

## Setup

> This directory is a placeholder. Frontend implementation is pending.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
